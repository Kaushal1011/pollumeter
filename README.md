# Pollumeter

## Air Pollution monitoring and parametric prediction based on contributing factors and regions

## Techstack
- Tensorflow
- Django
- GraphQL
- PostgreSQL
- ReactJS
- Apollo
- Recharts

## Starting Pollumeter
```sh
cd pollumeter
python manage.py runserver
// start backend

cd ..

cd pollumeter-fe/
npm i //install add frontend packages
npm start // start frontend 
```
